# Employee Management System

## ProDigy InfoTech - Full Stack Web Development Internship

### Task 02

This project is a simple Employee Management System developed as part of the ProDigy InfoTech Full Stack Web Development Internship.

## Features

- 🔐 Admin Login Authentication
- 👨‍💼 Employee Dashboard
- ➕ Add Employee
- 📋 View Employee List
- ✏️ Edit Employee Details
- 🗑️ Delete Employee
- 🚪 Logout Functionality
- 💻 Responsive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/sukrti71-hub/PRODIGY_FS_02.git
```

2. Install dependencies

```
npm install
```

3. Start the backend server

```
node backend/server.js
```

4. Open the frontend

```
frontend/index.html
```

or use Live Server in VS Code.
